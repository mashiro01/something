# php伪协议总结

在php中提供了这几种伪协议

|协议|用处|
|:--:|----|
|file://|访问本地文件系统|
|http:// </br> https://|访问 HTTP(s) 网址|
|ftp(s)://|访问 FTP(s) URLs|
|php:// </br> |访问各个输入/输出流（I/O streams）|
|zlib:// </br> bzip2:// </br> zip://| 压缩流 |
|data://|数据（RFC 2397）|
|glob://|查找匹配的文件路径模式|
|phar://|PHP 归档|
|ssh2://|Secure Shell 2|
|rar://|RAR|
|ogg://|音频流|
|expect://|处理交互式的流|
